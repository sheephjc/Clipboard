Clipboard 演示说明

              
Version: 2.0.2

              
支持文本、图片、文件历史记录、收藏筛选、图片 OCR、快速复制和删除操作。

              

                
              

            


            

              
识别结果

              

              

                
                
              

            

          

          
适用于所有人，尤其是经常查询历史剪贴板内容的用户。

          
例如写论文需要调用各个图片，或是需要来回处理部分文本。

        

      


      

        

          

            
为想要查询曾经复制内容的人准备。

            
轻量运行，默认驻留右下角任务栏托盘。关掉窗口后仍可继续监听，需要时再从托盘打开。

          

          

            

              文本
              
记录普通文本、富文本内容，编辑后可以重新复制。编辑区支持上下标，支持快捷键。

            

            

              图片
              
记录截图和复制的图片。支持图片文字识别。

            

            

              文件
              
文件和文件夹复制会进入“其他”分类，以路径的形式保存。

            

            

              功能丰富
              
支持开机自启，支持自动清理，支持收藏和日期筛选。

            

          

        

      


      

        

          

            
三步开始。

          

          

            

              1
              
下载压缩包并解压到常用位置。

            

            

              2
              
运行 Clipboard.exe，托盘图标出现后即开始记录。历史数据会在首次运行后生成到程序目录旁的 data 文件夹。

            

            

              3
              
如果更新到新版本后想要保留之前的记录，只需下载新的压缩包后将 data 文件夹移至同一目录下即可。

            

          

        

      


      

        

          

            
所有记录只存于该文件夹中。

            
所有复制的内容缓存默认保存在程序目录旁的 data 文件夹。不会对C盘有占用。

            
不涉及任何对其他隐私的侵犯和读取。
